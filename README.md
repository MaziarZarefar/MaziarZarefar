Web Developer
-------------

Hello! I'm a self-taught front-end developer with a strong passion for creating modern and responsive web applications.

My programming journey began in 2020, when I started learning Android development using Java. While I gained a solid foundation in coding through that experience, I eventually realized that my true interest lies in web development—particularly in front-end design and user experience.

I followed a structured roadmap and began by learning the core technologies: HTML, CSS, and JavaScript. As I advanced, I transitioned into using React.js, which has become my main framework of focus. I continue to work with React on a daily basis and actively follow the latest updates and trends in the ecosystem to keep my skills up-to-date and aligned with current industry standards.

My goal is to build visually engaging, user-friendly websites that not only look great but also provide real value to users.
